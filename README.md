# Student Grade Manager – Java

## Description
A Java-based student grade management system that demonstrates object-oriented programming, file I/O, and data management.

## Features
- Add students
- Add grades (0-100)
- Calculate individual averages, highest, lowest grades
- Class statistics and grade distribution
- File persistence (students.txt)

## How to Run Locally
```bash
javac Student.java GradeManager.java Main.java
java Main
